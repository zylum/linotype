# Linotype v0.6

v0.6 adds **LinoLoop**, a thin wrapper over executor briefs that can run a single galley or an ordered release list.

## What changed

### LinoLoop command

```bash
cli/linoloop <galley-name>
cli/linoloop <release-id>
```

Behavior:
- Generates executor brief via `cli/linotype exec opencode <galley>`
- Uses a loop runner (default `ralph`) when installed
- Falls back to printing the brief for manual paste if no runner is available

### Release format

Release file:
- `docs/work/releases/<release-id>/galleys.txt`

Rules:
- one galley ID per line
- blank lines allowed
- comment lines start with `#`

### Environment

- `LINOLOOP_AGENT` (default `opencode`)
- `LINOLOOP_MAX_ITER` (default `50`)
- `LINOLOOP_RUNNER` (default `ralph`)
